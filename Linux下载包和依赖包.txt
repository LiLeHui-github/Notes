sudo apt-get update
sudo apt-get install apt-rdepends

    下载包
    apt-get download <package_name>

说明
    查找包的所有依赖
    apt-rdepends <package_name>

    查找包的具体版本
    apt-cache policy <package_name>

    查询已安装的包
    dpkg -l | grep <package_name>

    下载包
    apt-get download <package_name>

    安装包
    sudo dpkg -i *.deb
