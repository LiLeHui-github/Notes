1.查看当前版本
    python -m pip show numpy
2.安装低版本 支持 win7 SP1
    python -m pip install "numpy==1.19.5" -i https://pypi.tuna.tsinghua.edu.cn/simple/
    python -m pip install "scipy==1.5.4" -i https://pypi.tuna.tsinghua.edu.cn/simple/
    python -m pip install "pip==20.3.4" -i https://pypi.tuna.tsinghua.edu.cn/simple/
    python -m pip install "setuptools==44.0.0" -i https://pypi.tuna.tsinghua.edu.cn/simple/
或重新安装
    python -m pip install --upgrade "numpy==1.19.5" -i https://pypi.tuna.tsinghua.edu.cn/simple/
    python -m pip install --upgrade "scipy==1.5.4" -i https://pypi.tuna.tsinghua.edu.cn/simple/
    python -m pip install --upgrade "pip==20.3.4" -i https://pypi.tuna.tsinghua.edu.cn/simple/
    python -m pip install --upgrade "setuptools==44.0.0" -i https://pypi.tuna.tsinghua.edu.cn/simple/
