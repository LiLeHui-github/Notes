
创建虚拟磁盘
qemu-img.exe create -f qcow2 D:/VirtualMachines/Systems/Kylin10SP1-General-2303-ARM64/kylin_arm64.qcow2 60G

下载 ARM64 UEFI 固件
http://releases.linaro.org/components/kernel/uefi-linaro/16.02/release/qemu64/QEMU_EFI.fd
保存到：D:/VirtualMachines/ISO/ARM/QEMU_EFI.fd

从 ISO 启动安装
qemu-system-aarch64.exe -m 16G -cpu cortex-a76 -smp 8 -M virt -bios D:/VirtualMachines/ISO/ARM/QEMU_EFI.fd -rtc base=localtime -display gtk,show-cursor=on -device virtio-gpu-pci -device nec-usb-xhci -device usb-tablet -device usb-kbd -drive if=none,file=D:/VirtualMachines/Systems/Kylin10SP1-General-2303-ARM64/kylin_arm64.qcow2,id=hd0,format=qcow2 -device virtio-blk-device,drive=hd0 -drive if=none,file=D:/VirtualMachines/ISO/ARM/Kylin-Desktop-V10-SP1-General-Release-2303-ARM64.iso,id=cdrom,media=cdrom -device virtio-scsi-device -device scsi-cd,drive=cdrom -netdev user,id=net0,hostfwd=tcp::2222-:22 -device virtio-net-pci,netdev=net0 --accel tcg,thread=multi

Windows挂载虚拟磁盘传输文件
Windows端操作
创建一个虚拟磁盘    qemu-img create -f raw share.img 100M
格式化这个虚拟盘    qemu-img convert -f raw -O vpc share.img share.vhd
双击 share.vhd Windows系统会自动挂载，格式化它为 FAT32
启动 QEMU 时带上这个共享盘
     -drive file=share.img,if=virtio,format=raw  # 共享盘

虚拟机端操作
新建挂载目录    mkdir -p /mnt/share
直接挂载        mount -t vfat /dev/vdb /mnt/share

快照管理
    创建：-c
    查看：-l
    恢复：-a
    删除：-d
创建快照
qemu-img snapshot -c 快照名 D:/VirtualMachines/Systems/Kylin10SP1-General-2303-ARM64/kylin_arm64.qcow2

查看所有快照
qemu-img snapshot -l D:/VirtualMachines/Systems/Kylin10SP1-General-2303-ARM64/kylin_arm64.qcow2

恢复到某个快照
qemu-img snapshot -a 快照名 D:/VirtualMachines/Systems/Kylin10SP1-General-2303-ARM64/kylin_arm64.qcow2

删除某个快照
qemu-img snapshot -d 快照名 D:/VirtualMachines/Systems/Kylin10SP1-General-2303-ARM64/kylin_arm64.qcow2
